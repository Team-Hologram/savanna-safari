# Savanna Safari

Next.js App Router project with 3D hero (fallback image), mock booking flow, and basic API routes.

## Quickstart

```bash
pnpm i   # or npm i / yarn
pnpm dev
```

## Tech
- Next.js 15 (App Router, TypeScript)
- TailwindCSS
- @react-three/fiber + drei (placeholder setup)
- Zustand store
- Simple API routes (/api/rides, /api/availability) powered by mock JSON
